# Dashboard de E-commerce no Brasil

Este dashboard foi desenvolvido durante meu curso de Power BI no Alura, com o objetivo de **analisar os pedidos de e-commerce no Brasil entre 01/2017 e 08/2018**.

## 🔎 Objetivo
O painel permite visualizar:
- **Total de pedidos** no período analisado.
- **Evolução mensal dos pedidos**, destacando tendências e sazonalidades.
- **Preferência de pagamento**, identificando quais métodos são mais utilizados.
- **Quantidade de produtos por categoria**, mostrando os segmentos com maior representatividade.

## 📈 Principais Insights
- Foram realizados **44,20 mil pedidos** no período.
- O **cartão de crédito** é o método de pagamento mais utilizado (74% dos pedidos).
- A categoria **Cama, Mesa e Banho** lidera em volume de produtos vendidos (4,34 mil).
- O volume de pedidos cresce de forma consistente, com destaque para **dezembro**, que alcançou **7,21 mil pedidos**.
